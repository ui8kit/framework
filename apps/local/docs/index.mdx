# UI8Kit Components

Welcome to the UI8Kit component documentation. This documentation is built using MDX and provides interactive examples of all UI8Kit components.

## Getting Started

UI8Kit provides a comprehensive set of React components with TypeScript support and consistent design system integration.

### Quick Example

<div className="example-preview" data-class="example-preview">
  <Button variant="primary">Hello World</Button>
  <Button variant="secondary">Secondary</Button>
</div>

```tsx
import { Button } from '@ui8kit/core'

export function App() {
  return (
    <div>
      <Button variant="primary">Hello World</Button>
      <Button variant="secondary">Secondary</Button>
    </div>
  )
}
```

## Design Principles

### Consistency
All components follow the same design patterns and use the UI8Kit design tokens.

### Accessibility
Components are built with accessibility in mind, following WCAG guidelines.

### Performance
Components are optimized for performance with tree-shaking support.

### TypeScript
Full TypeScript support with comprehensive type definitions.
