# UI8Kit Documentation

Welcome to the UI8Kit documentation. UI8Kit provides React components with TypeScript support and generates semantic static HTML.

## Quick Start

```tsx
import { Button, Card, Stack } from '@ui8kit/core'

export function App() {
  return (
    <Stack gap="4">
      <Card>
        <Button variant="primary">Get Started</Button>
      </Card>
    </Stack>
  )
}
```

## Features

- **React Components** — Type-safe components with utility props
- **Static Generation** — Generate semantic HTML without JavaScript
- **Design Tokens** — Consistent theming with Shadcn design tokens
- **Tailwind Integration** — Works with Tailwind CSS
