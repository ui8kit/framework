# Button

The Button component provides interactive buttons with various styling options.

## Basic Usage

<div className="example-preview" data-class="example-preview">
  <Button>Click me</Button>
</div>

```tsx
<Button>Click me</Button>
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'destructive'` | `'default'` | Button variant |
| `size` | `'xs' \| 'sm' \| 'default' \| 'md' \| 'lg' \| 'xl'` | `'default'` | Button size |
| `disabled` | `boolean` | `false` | Disabled state |
| `children` | `ReactNode` | - | Button content |

### CSS Classes

- `button` - Base button class
- `button-{variant}` - Variant-specific styling
- `button-{size}` - Size-specific styling

### Data Attributes

- `data-class="button"` - Semantic class identifier
