# 📱 Мобильная адаптивность MainLayout

## ✅ Что добавлено

Полная мобильная адаптивность для MainLayout с правильным переводом в одноколончный режим на мобайлах.

## 🔧 Как это работает

### 1. Grid медиа-класс

```tsx
<Grid 
  grid="cols-3"                           // Base: 3 колонки
  gap="8"                                 // Spacing
  className="md:grid-cols-3 grid-cols-1"  // Tailwind медиа-запросы
>
```

**Результат:**
- `md:grid-cols-3` — На планшетах и выше (≥768px): 3 колонки
- `grid-cols-1` — На мобайлах (<768px): 1 колонка

### 2. Content медиа-класс

```tsx
<Stack 
  col="span-2"                            // Base: 2 из 3 колонок
  className="md:col-span-2 col-span-1"    // Tailwind медиа-запросы
>
```

**Результат:**
- `md:col-span-2` — На планшетах и выше: 2 колонки (66%)
- `col-span-1` — На мобайлах: 1 колонка (full width)

### 3. Sidebar медиа-класс

```tsx
<Stack
  col="span-1"                            // Base: 1 из 3 колонок
  className="md:col-span-1 col-span-1"    // Tailwind медиа-запросы
>
```

**Результат:**
- `md:col-span-1` — На планшетах и выше: 1 колонка (33%)
- `col-span-1` — На мобайлах: 1 колонка (full width, под контентом)

## 🎯 Медиа-запросы (Tailwind)

| Класс | Desktop (md) | Mobile |
|-------|---|---|
| `md:grid-cols-3` | 3 колонки ✓ | игнорируется |
| `grid-cols-1` | игнорируется | 1 колонка ✓ |
| `md:col-span-2` | 2 колонки ✓ | игнорируется |
| `col-span-1` | игнорируется | 1 колонка ✓ |

## 💡 Как работают Tailwind медиа-префиксы

```
md:     → Media query: @media (min-width: 768px)
        → Применяется на md и выше (768px, 1024px, 1280px, 1536px)

Без префикса → применяется всегда (мобайл вверх)

Результат: мобайл-фёрст подход
```

## 🔄 Order для sidebar-left

Даже при `sidebar-left` режиме на мобайле всё работает правильно:

```tsx
order={isSidebarLeft ? "1" : "2"}  // Сайдбар

// Результат:
// Desktop (with-sidebar): Сайдбар справа (order-2)
// Desktop (sidebar-left): Сайдбар слева (order-1)
// Mobile: Всегда под контентом, но order-1 или order-2 зависит от режима
```

## ✨ Особенности реализации

✅ **Комбинированный подход**
- ui8kit пропсы: базовая структура (`grid="cols-3"`, `col="span-2"`)
- Tailwind className: медиа-запросы для адаптивности

✅ **Мобайл-фёрст**
- На мобайле: 1 колонка (по умолчанию)
- На десктопе: 3 колонки (переопределяется md: префиксами)

✅ **Все режимы поддерживаются**
- `full` — всегда полная ширина
- `with-sidebar` — 2:1 на десктопе, 1:1 стакирование на мобайле
- `sidebar-left` — 1:2 на десктопе, 1:1 стакирование на мобайле

✅ **Data-class для CSS генерации**
- Все элементы имеют data-class для автоматической генерации CSS
