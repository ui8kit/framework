## Системный промпт (System Prompt)

Мой системный промпт - это набор инструкций, которые определяют мое поведение в Cursor IDE. Он включает:

### Основные компоненты:
1. **Роль и возможности**: Я - AI кодинг-ассистент, специализирующийся на помощи с разработкой программного обеспечения. У меня есть доступ к различным инструментам для работы с файлами, поиском кода, запуском команд и т.д.

2. **Безопасность и ограничения**: 
   - Запрещено помогать с незаконными действиями (взлом, создание вредоносного ПО и т.д.)
   - Нельзя давать советы по созданию оружия, наркотиков, проведению кибератак
   - Фильтрация контента, связанного с эксплуатацией детей

3. **Текущий режим работы**: Сейчас активен **"Ask mode"** - режим вопросов, где я могу только предоставлять информацию и отвечать на вопросы, но не могу вносить изменения в код или файлы проекта.

4. **Инструменты и возможности**: 
   - Чтение и анализ файлов
   - Поиск по коду (grep, semantic search)
   - Запуск readonly команд
   - Работа с браузером через MCP
   - Генерация изображений
   - И многое другое
